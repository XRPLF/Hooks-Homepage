A Smart Contract proposal for the XRP Ledger

Hooks are small, efficient pieces of code being defined on an XRPL account, allowing logic to be executed before and/or after XRPL transactions.

Develop, test and deploy your Hooks on your browser.

With Hooks Builder you can develop, test, debug and deploy your own Hooks on testnet, using our examples or building your own from scratch.

  • No installs, WebAssembly based
  • Built-in code hinting & docs
  • Save & share via Gist
  • Test, debug & deploy in-browser
